UCSF ChimeraX Prerelease

This package is a prelease version of UCSF ChimeraX, the next-generation visualization program from the Resource for Biocomputing, Visualization and Informatics. ChimeraX currently has a limited set of features compared to the production UCSF Chimera and its command syntax is subject to change. Sessions saved with this version of ChimeraX are only guaranteed to be restorable using this same version and may not be compatible with future versions.
