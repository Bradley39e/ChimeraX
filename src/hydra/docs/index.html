Commands

open filename,...|database-id,... [fromDatabase PDB|EMDB|EDS] - open files
save filename.[jpg|png|ppm|bmp] [width w] [height h] [supersample s] - save an image
save filename.hy - save a session file
save filename.mrc - save a density map file
close [model-spec|all] - close models
scene add|show|delete [id] - remember views
volume - Control density map display.
vop bin|falloff|gaussian|maximum|threshold map-ids - Filter density maps.
vop bin map-ids [binSize b] [subregion i0,j0,k0,i1,j1,k1] [step s|si,sj,sk] [modelId id] - bin map taking average values in b x b x b cells.
vop falloff map-ids [iterations i] [inPlace false|true] [subregion i0,j0,k0,i1,j1,k1] [step s|si,sj,sk] [modelId id] - smooth mask edges.
vop gaussian map-ids [sDev s] [valueType t] [subregion i0,j0,k0,i1,j1,k1] [step s|si,sj,sk] [modelId id] - Gauassian smoothing.
vop maximum map-ids [onGrid map-id] [boundingGrid true|false] [scaleFactors s,...] [inPlace false|true] [gridSubregion i0,j0,k0,i1,j1,k1] [gridStep s|si,sj,sk] [subregion i0,j0,k0,i1,j1,k1] [step s|si,sj,sk] [modelId id] - maximum value of 2 or more maps.
vop threshold map-ids [minimum v] [set min] [maximum v] [setMaximum max] [subregion i0,j0,k0,i1,j1,k1] [step s|si,sj,sk] [modelId id] - replace map values below or above specified values with new values.
fitmap - Fit molecules in maps.
molmap - Calculated simulated density maps from molecules.
show atom-spec [atoms] [ribbon] [only] - display specific atoms or ribbons.
hide atom-spec [atoms] [ribbon] [only] - hide specific atoms or ribbons.
cycle models [wait frames] [frames n] [bounce] [stop] - display models one at a time in sequence.
color atom-spec color [atoms] [ribbon] - color atoms or ribbons.
turn x|y|z|x,y,z angle [frames] - rotate models (achieved by moving camera).
freeze - stop multiframe motion started by turn command
align atom-spec ref-atoms [move molecules|chains|residues|atoms|false] [each molecule|chain] [same n] [report_matrix true|false] - align molecules to minimize RMSD.
surface atom-spec [probeRadius r] [gridSpacing s] [waters] - Show a solvent excluded molecular surface.
area atom-spec [probeRadius r] - Report solvent accessible surface area.
blast chain-spec [sequence seq] [uniprot id] [dropSimilarChains rmsd] [dropShortChains nlen] [blastProgram path] [blastDatabase path] - Find pdb entries with matching a sequence, load and align them.
windowsize [width height] - Set or report graphics window size in pixels.
camera [mono|stereo|oculus] [fieldOfView degrees] [eyeSeparation millimeters] [screenWidth millimeters] [sEyeSeparation scene-dist] [middleDistance] [depthScale f] [report] - Adjust or report camera settings.
lighting [direction x,y,z] [color r,g,b|name] [fillDirection x,y,z] [fillColor r,g,b|name] [ambientColor r,g,b|name] [fixed true|false] [shadows true|false] [qualityOfShadows normal|fine|finer|coarse|shadow-map-size] [multiShadow n] - Change lighting parameters. Direction vectors should have unit length.
silhouette on|off [thickness pixels] [color r,g,b|name] [depthJump j] - Enable silhouette edges.
material [ambientReflectivity r] [diffuseReflectivity r] [specularReflectivity r] [exponent e] - Change surface reflectivity parameters.
ambient model-spec [fineness f] [light l] [dark d] [bin_size b] [texture true|false] [gaussian true|false] - Color molecule, surface, or map based on density to approximate ambient occlusion darkening.
device [SpaceNavigator|OculusRift|LeapMotion] [on|off] [fly true|false] [mode position|velocity|chopsticks] [scale f] [lensCorrection true|false] - turn on or off input and output devices.
movie record|encode options - record videos.
movie record [directory path] [pattern filepat] [format ppm|png|jpg] [size w,h] [supersample n] [limit frames] - start capturing images for a movie.
movie encode [output-path] [format h264|mov|avi|mp4|mp2|mpeg|wmv] [quality good|low|fair|medium|high|higher|highest] [qscale n] [bitrate kbits-per-sec] [framerate r] [resetMode clear|keep] [roundTrip false|true] - save captured images as a video.
wait [frames] - wait a specified number of frames or if no frames specified until motion stops before going to next command.
