Command: phenix

The phenix command is provided by the PhenixUI bundle. It interfaces with a local installation of Phenix. See also: Volume Viewer

• phenix douse  map-model  nearModel  atomic-model  [ resolution  r ] [ keepInputWater  true | false ] [ farWater  true | false ] [ mapRange  distm ] [ residueRange  distr ] [ verbose  true | false ] [ phenixLocation  folder ] [ block  true | false ]

Run the Phenix program douse to place water molecules in cryo-EM density. The density map and corresponding atomic model should be already open and specified as the  map-model   atomic-model, respectively.

Remaining options

  • resolution – map resolution (default 3.0 Å)
  • keepInputWater (default true) – whether to keep pre-existing waters instead of deleting them
  • farWater (default false) – if keepInputWater is true, whether douse should use the pre-existing waters; since douse only places waters near existing atoms, ignoring pre-existing waters limits the results to waters that interact directly with the rest of the structure (farWater false, default)
  • mapRange – show only map within distm (default 8.0 Å) of water atoms; specifying the distance as 0 (zero) turns off hiding by distance
  • residueRange – show residues with any atom within distr (default 5.0 Å) of water atoms
  • verbose (default false) – whether to echo all douse messages to the Log
  • phenixLocation – where Phenix is installed (default is the system Applications folder); giving folder as the word browse opens a file browser window for locating the folder interactively
  • block – whether to wait for the calculation to finish before starting to execute any subsequent commands, default true when the command is being executed in a command script and/or in nogui mode, otherwise false
