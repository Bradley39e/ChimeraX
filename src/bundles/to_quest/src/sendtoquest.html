Send to Quest

This tool sends a scene to an Oculus Quest VR headset to be viewed by the LookSee application. See instructions for how to install LookSee on a Quest 2 or Quest Pro headset.

The scene must be simple enough for the Quest 2 or Quest Pro graphics to handle without flickering. If the room is shown (with pass-through video) the scene should have less than 700000 million triangles and with room not shown (black background) less than 900000 triangles.

In the LookSee app clicking the A or X hand-controller button will show the user interface, and clicking "Open new files" will make new files sent from ChimeraX automatically display. There may be up to a 5 second delay displaying a new file.

Setup

To send the scene the computer must have the adb (Android Debug Bridge) command line program installed.

The scene shown in ChimeraX can be sent wirelessly or over a USB cable connecting the computer to the VR headset. To set up wireless transfer follow these steps

  1. Plug in Quest to computer with usb cable.
  2. Click Allow Access in Quest headset.
  3. Check Quest IP address in headset clicking the wifi symbol in the Quest, then clicking the connected wifi name, then scroll down and click Advanced.
  4. Use adb command "adb tcpip 5555" from a shell on the computer.
  5. Use adb command "adb 192.168.0.135" to specify the Quest IP address.
  6. Disconnect usb cable from Quest.
  7. Optionally, check connected devices with "adb devices"
