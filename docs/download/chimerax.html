UCSF ChimeraX Prerelease

November 25, 2015

UCSF ChimeraX is the next-generation visualization program from the Resource for Biocomputing, Visualization, and Informatics at UC San Francisco, following Midas, MidasPlus, and Chimera.

Features

Compared to Chimera, ChimeraX has better graphics (e.g. interactive ambient shadows), handles large structures (millions of atoms) much faster, and has a modern single-window user interface (see short list of advantages). ChimeraX will have a web site of available extensions written by others using documented programming interfaces.

Missing Features

ChimeraX currently has a limited set of features compared to the production Chimera. It has almost no user-interface dialogs, no structure alignment, no sequence tools, no trajectory playback, no distance measurements, no hydrogen-bond or clash detection, etc. We will be adding these in the future.

Compatibility

ChimeraX is not backward compatible with Chimera, although many aspects will be familiar from Chimera.

Linux Notes

The Linux download will not work on Linux distributions besides Debian 8 because of many missing system libraries. It is possible to install all the needed libraries. We have used it on Ubuntu 15.04 after installing a handful of missing libraries.
