UCSF ChimeraX Prerelease

November 11, 2015

UCSF ChimeraX is the next-generation visualization program from the Computer Graphics Lab at UC San Francisco, following Midas, MidasPlus, and Chimera.

Features

ChimeraX has better graphics (e.g. interactive ambient shadows), handles large structures (millions of atoms) much faster, has a modern single-window user interface, and will have a website describing available extensions written by others using documented programming interfaces.

Compatibility

ChimeraX is not backward compatible with Chimera, although many aspects will be familiar from Chimera.

Linux Notes

The Linux download will not work on Linux distributions besides Debian 8 because of many missing system libraries. It is possible to install all the needed libraries. We have used it on Ubuntu 15.04 after installing a handful of missing libraries.
