Protein Structures Predicted by AlphaFold

Theoretical protein structures predicted by the AlphaFold AI system can be fetched by ChimeraX from the AlphaFold Database. Users of this resource should cite:

Highly accurate protein structure prediction with AlphaFold. Jumper J, Evans R, Pritzel A, et al. Nature. 2021 Jul 15. doi: 10.1038/s41586-021-03819-2.

The AlphaFold Database does not cover all sequences in UniProt, so there may not be a prediction available for a given protein chain. It is also important to realize that the predicted structures vary in confidence levels and should be interpreted with caution. See also: Modeller Comparative

What structures to fetch can be specified in two main ways.

  1. by UniProt accession number, for example:
    open p29474 from alphafold
    – OR –
    alphafold fetch p29474
  2. by chain(s) of an experimentally determined atomic model already open in ChimeraX, for example:
    alphafold match #1/B,D
    alphafold match #3 trim false

    Giving a model number specifies all of its protein chains. Specifying one or more chains of an existing model means to:

    1. retrieve the prediction, if any, from the AlphaFold Database for each chain's UniProt accession number identified as described below
    2. for each available predicted structure, make the chain ID the same as the corresponding chain of the existing model
    3. superimpose each predicted structure on the corresponding chain of the model using matchmaker defaults
    4. if the option colorConfidence is true (default), color the predicted structures by the pLDDT confidence measure in the B-factor field:
      • 90
        to 100
        – high accuracy expected
      • 70
        to 90
        – expected to be modeled well (a generally good backbone prediction)
      • 50
        to 70
        – low confidence, treat with caution
      • 0
        to 50
        – should not be interpreted, may be disordered
      ...in other words, using palette 0,red:50,orange:70,yellow:90,cornflowerblue:100,blue

    UniProt accession numbers may be included in the experimental structure's input file; when present, these are given in DBREF records in PDB format and in struct_ref and struct_ref_seq tables in mmCIF. With the alphafold match option search true (default), if the accession number for a chain is not provided in the file, or is provided but not found in the AlphaFold Database, BLAT is used to search the 21 proteomes in the database for the closest match. The closest sequence match for which a prediction is available will be retrieved, as long as the sequence identity is at least 25%. With search false, however, only the experimental structure's input file will be used as a potential source of UniProt accession numbers.

    The trim option (default true) indicates whether to trim the predicted protein structures to the same residue ranges as used in the experiment to determine the known structure. These ranges are given in DBREF records in PDB format and in struct_ref and struct_ref_seq tables in mmCIF. Use trim false to include the full UniProt sequences, which could be longer.

The alphafold command also has an ignoreCache option (default false) with the same meaning as for the open command. Namely, ignoreCache true forces fetching the database entry even if it has already been cached locally.
