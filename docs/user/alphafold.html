Protein Structures Predicted by AlphaFold

Theoretical protein structures predicted by the AlphaFold AI system can be fetched by ChimeraX from the AlphaFold Database. Users of this resource should cite:

Highly accurate protein structure prediction with AlphaFold. Jumper J, Evans R, Pritzel A, et al. Nature. 2021 Jul 15. doi: 10.1038/s41586-021-03819-2.

The AlphaFold Database does not cover all sequences in UniProt, so there may not be a prediction available for a given protein chain. It is also important to realize that the predicted structures vary in confidence levels and should be interpreted with caution. See also: Modeller Comparative

The structures to fetch can be specified in two main ways with the open command:

  1. by UniProt ID or accession code, for example:
    open P29474 from alphafold
  2. by model number of a structure already open in ChimeraX, for example:
    open #3 from alphafold trim true

    Specifying an existing model number means to:

    1. search the AlphaFold Database using the UniProt identifier (if any) provided for each protein chain in the mmCIF or PDB file of that model
    2. for each available predicted structure, make the chain ID the same as the corresponding chain of the existing model
    3. superimpose each predicted structure on the corresponding chain of the model using matchmaker defaults
    4. color the predicted structures by the pLDDT confidence measure in the B-factor field:
      • 90
        to 100
        – high accuracy expected
      • 70
        to 90
        – expected to be modeled well (a generally good backbone prediction)
      • 50
        to 70
        – low confidence, treat with caution
      • 0
        to 50
        – should not be interpreted, may be disordered

    The trim option indicates whether to trim the predicted proteins to the residue ranges in the PDB SEQRES records (or mmCIF equivalent) of the experimental structure. The alternative is to include the full UniProt sequences, which could be longer.
