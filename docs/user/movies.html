Making Movies

A simple spin movie can be created using the “video recorder” Graphics icon:

Otherwise, the movie command can be used to start recording, stop recording, and encode the saved images as a movie file. During recording, the scene can be manipulated interactively and commands can be entered. However, for reproducibility and smoother results, often a ChimeraX command file (script) integrating movie with other commands will be used instead.

Below are some of the commands likely to be used for scripting continuous motion and other movie content. For general appearance, see also: making images, Graphics Toolbar

  • 2dlabels - static annotation with 2D text and symbols
  • clip - control clipping planes
  • cofr - set center of rotation
  • crossfade
  • move - translate models
  • movie - record image frames and assemble them into a movie file
  • perframe - similar to Chimera perframe
  • roll - rotate models continuously
  • stop - halt ongoing motions
  • turn - rotate models
  • view - adjust the view to specified items; save/restore named views
  • vop morph - morph (interpolate) between two or more maps
  • vseries - similar to Chimera vseries
  • wait - update the display and enforce ordered execution of commands in scripts
  • windowsize - set pixel width and height of the graphics window
  • zoom - change the apparent size of the view
