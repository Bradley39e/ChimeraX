Tool: Segmentations

The Segmentations tool allows interactively segmenting medical image data plane by plane, where medical image refers to volume data opened from the DICOM, NIfTI (.nii), and/or NRRD (.nrrd) formats. Segmentations are additional volume models displayed as surfaces that delineate regions of arbitrary shape within the image dataset; they typically represent anatomic structures or lesions such as tumors. Multiple segmentation models can be defined from the same image dataset.

For semiautomated segmentation of cryo-electron microscopy density maps, see the Segment Map tool instead. See also: Download DICOM, Volume Viewer, Map Eraser, dicom, color zone, volume mask, segmentation, ChimeraX DICOM Reference, ChimeraX for Medical Image Analysis

The Segmentations tool can be started from the Medical Imaging section of the Tools menu (more...). The medical-image display is automatically changed to a four-panel view with axial, coronal, and sagittal slices along with a 3D rendering (see slice navigation).

Creating and Editing Segmentations

Segmentation models (if any) are listed in the dialog. Clicking Add creates a new segmentation model, and any segmentation in the list can be can be chosen (highlighted) by mouseclick. The currently highlighted segmentation is the one that will be populated using the segmentation cursor in the slice views.

Clicking Remove deletes the chosen segmentation. Edit allows editing the metadata associated with the segmentation model, and Save saves it as DICOM with the SEG modality.

The segmentation cursor is shown as a red circle; left mouse click-drag sweeps out an area in the currently displayed data plane, whereas additionally pressing Shift erases the corresponding area from the segmentation. Shift-scroll adjusts the segmentation cursor size.
