Tool: ViewDockX

ViewDockX is a tool for the interactive analysis of molecular docking results. Users can click through a list of docked compounds (or different poses of the same compound) to view them individually in the context of a binding site. Results can be sorted, rated, and saved, and scores or other numerical descriptors can be plotted. See also: hbonds, clashes, an example file of compounds docked to H-ras (the protein part of PDB entry 121p)

After one or more files of docking results have been opened, choosing ViewDockX from the Tools menu shows a table listing the different compounds or poses. The ViewDockX table and associated plots can be manipulated like other panels in ChimeraX (more...).

ViewDockX Table

In the table of docking results, the rows are different compounds or different positions of a compound, and the columns show additional data read from the input, typically including compound names and docking scores. The first three columns are generated automatically:

  • S - show/hide checkboxes
  • RATING - star rating
  • ID - model identification number, which can be clicked to show the model and hide all others listed in the table

Any additional columns that appear when the tool is first started directly reflect information read from the input file(s). Available columns of information can be hidden from the table and reshown by clicking the arrowhead near the top right corner of the table and using the resulting checkbox menu. Clicking the header of a column sorts by the values in that column.

Buttons above the table:

  • Show All Structures - show structures for all table rows
  • Plot - plot one or more numerical descriptors for all table rows
  • Save... - for all table rows, save structures and descriptors (including current star ratings) to a Mol2 file; the file can be reopened later for further analysis with ViewDockX
  • Close - close any structures with star ratings no greater than the specified number of stars, and remove their rows from the table

H-BONDS between a docked compound and a receptor structure can be calculated as described for the hbonds command and included as an additional column.

Plotting

Clicking the Plot button near the top left of the ViewDockX table opens another window for plotting the table entries by their numerical descriptors. In the plotting window, one of the numerical descriptors can be chosen for the Sort order, and all or a subset of them as values to Plot. Multiple plotting windows can be shown for the same set of docking results.

Pausing the cursor over a plotted point shows the corresponding model number, name, and value. Clicking the point shows only that model (amongst the models in the set of docking results).
