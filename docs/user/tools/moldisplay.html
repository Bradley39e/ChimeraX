Tool: Molecule Display

The Molecule Display tool provides a series of icons for changing the display of atomic models or their selected parts with a single click. Like other tools, this interface can be opened from the Tools menu and closed by clicking the small “x” near its top right corner, or shown/hidden with the toolshed command. See also: Graphics icons, Mouse Modes for Right Button icons

Clicking an icon shows the executed command(s) in the Log. The selAtoms specifier indicates the current selection, if any, otherwise all atomic models.
