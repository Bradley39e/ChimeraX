Tool: Log

The Log tool echoes commands that have been executed, with any truncations expanded and links to show their help pages. The Log also shows messages and other output information. Like other tools, this interface can be opened from the Tools menu (more...).

The log command performs several functions related to the Log, including clearing its contents or saving them to an HTML file.

See also: Command History, usage, ChimeraX browser
