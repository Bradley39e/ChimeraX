Tool: Mouse Modes for Right Button

The Mouse Modes for Right Button toolbar provides a series of icons for changing the assignment of the right mouse button with a single click. Like other tools, this interface can be opened from the Tools menu.

With the related mousemode command, functions can be assigned to all mouse buttons (not just the right) and to scrolling, alone and in combination with modifier keys. In the case of a touchpad or single-clicker mouse, middle- and right-button emulation are normally provided by the alt and command keys, respectively. See also: Density Map Toolbar, Graphics Toolbar, Molecule Display Toolbar
