Of course, many more commands exist but lack detailed user documentation, and may be experimental and/or in flux. A subset are listed below, and the help command can be used to show brief descriptions of specific commands or all commands in the Log (e.g., help style or help all). Some commands are much the same as in Chimera1. Some may be removed and cease to exist in Chimera2.

  • open - open local file or fetch from repository; first model is #1; default for 4-character ID is to fetch mmCIF; use pdb: prefix to fetch PDB, emdb: prefix to fetch from EMDB
  • close - close model
  • list - list current models
  • color
  • display/~display
  • ribbon/~ribbon
  • style (with options stick, ball, sphere)
  • lighting (with options including soft for ambient occlusion, flat for ambient-only, full with shadows, default)
  • set (with options including bgColor and silhouette)
  • delete
  • exit or quit
  • sym
  • sasa
  • buriedarea
  • contacts

  • vseries

  • crossfade

  • roll
  • stop - stop motion, like Chimera1 freeze
  • wait
