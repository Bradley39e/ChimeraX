      • differences vs. Chimera
    • 2dlabels - static annotation with 2D text and symbols
    • alias - define a command alias (shortcut or composite action)
    • camera - set mono or stereo viewing and related parameters
    • cartoon or ribbon - create cartoons, set cartoon parameters
    • clip - control clipping planes
    • close - close models
    • cofr - set center of rotation
    • color, rainbow - color atoms/bonds, cartoons, molecular surfaces
    • delete - delete atoms
    • exit or quit - exit from ChimeraX
    • lighting - report or adjust lighting parameters
    • log - clear, save to file, and other actions related to the Log
    • material - report or adjust material properties for light reflection
    • move - translate models
    • mousemode - report or adjust mouse button assignments
    • move - translate models
    • movie - record image frames and assemble them into a movie file
    • open - open local file or fetch from repository
    • position - set location and orientation of camera, models by input matrix
    • pwd - print working directory to the Log
    • roll - rotate models continuously
    • save - save image or session
    • select - select items (models or their parts) for subsequent operations
    • set - set background color, silhouette parameters, subdivision level
    • show, hide - show/hide atoms, bonds, cartoons, models
    • stop - halt ongoing motions
    • style - set display style of atoms and bonds
    • surface - create and show/hide molecular surfaces
    • sym - build multimers using mmCIF assembly information
    • transparency - adjust transparency of atoms/bonds, cartoons, molecular surfaces
    • turn - rotate models
    • version - report ChimeraX version
    • view - adjust the view to specified items; save/restore named views
    • wait - update the display and enforce ordered execution of commands in scripts
    • windowsize - set pixel width and height of the graphics window
    • zoom - change the apparent size of the view

    Additional commands exist but lack detailed documentation; several are listed below. Some are nearly the same as the corresponding Chimera commands, except for target-specification differences. Try usage command (for example, usage set) to show a brief description, help command to show full documentation, if available. Commands and their syntax are potentially subject to change.

    • align - superimpose sets of atoms
    • sasa - calculate solvent-accessible surface area
    • buriedarea - calculate buried surface area
    • contacts - draw interchain contact diagram based on buried surface areas
    • volume - many options for density-map display; almost the same as Chimera volume
    • vop - almost the same as Chimera vop
    • vseries - almost the same as Chimera vseries
    • molmap - almost the same as Chimera molmap
    • fitmap - almost the same as Chimera fitmap
    • mlp - calculate molecular lipophilicity potential (MLP) for peptides/proteins
    • crossfade
    • perframe - similar to Chimera perframe
    • toolshed or ts - start, show/hide, and otherwise manage ChimeraX Tools
    • list - list current models
    • pdbimages - batch-render images of molecular assemblies defined in mmCIF files (see examples)
    • oculus
