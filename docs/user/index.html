Of course, many more commands exist but lack detailed user documentation, and may be experimental and/or in flux. A subset are listed below, and the help command (e.g., help style) can be used to show their brief descriptions in the Log. Some are very similar to the corresponding commands in Chimera1. Some may be removed and cease to exist in Chimera2.

  • open - open local file or fetch from repository; first model is #1; default for 4-character, PDB-like ID is to fetch mmCIF; use pdb: prefix to fetch PDB, emdb: prefix to fetch from EMDB
  • close - close model
  • list - list current models
  • color
  • display/~display
  • ribbon/~ribbon
  • style (with style options stick, ball, sphere)
  • delete
  • exit
  • help
  • movie
  • crossfade
  • move
  • turn
  • roll
  • stop - stop motion, like Chimera1 freeze
  • wait
  • 2dlabels
  • perframe
  • volume
  • vop
  • vseries
  • sym
  • contacts
  • save
  • light (with options including soft for ambient occlusion, flat for ambient-only, full with shadows, default)
  • set (with options including bgColor and silhouette)
  • material
  • camera
  • oculus
  • window
