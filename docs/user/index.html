UCSF Chimera2 User Guide

UCSF Chimera2 is the successor program to UCSF Chimera, here termed “Chimera1.”

  • General:
    • advantages vs. Chimera1
    • Quick Start Guide
  • Tools:
  • Commands:
    • color - color atomic models, cartoons, molecular surfaces
    • colordef - define a color name
    • open - open local file or fetch from repository
    • select - select items (models or their parts) for subsequent operations
    • style - set display style of atoms and bonds
    • surface - create and display molecular surfaces
    • transparency - adjust transparency of atomic models, cartoons, molecular surfaces
    • view - bring all displayed atoms or specified atoms into view

Many more commands exist, but are not yet fully documented. Some are experimental and could be changed significantly or omitted from later versions of Chimera2. Several current commands are listed below. The usage command can be used to show a brief description of a command in the Log (for example, usage set). The help command (for example, help style) shows the full manual page for a command if available, otherwise just the usage. Some commands are much the same as the corresponding Chimera1 commands, except for the specification differences noted above.

  • list - list current models
  • close - close model
  • display/~display
  • ribbon/~ribbon
  • lighting (with options including soft for ambient occlusion, flat for ambient-only, full with shadows, default)
  • set (with options including bgColor and silhouette)
  • delete
  • exit or quit
  • sym
  • sasa
  • buriedarea
  • contacts
  • volume - same as Chimera1 volume
  • vop - same as Chimera1 vop
  • vseries - same as Chimera1 vseries
  • molmap - same as Chimera1 molmap
  • fitmap - same as Chimera1 fitmap
  • movie - same as Chimera1 movie
  • crossfade
  • move - similar to Chimera1 move
  • turn - similar to Chimera1 turn
  • roll - similar to Chimera1 roll
  • stop - stop all motion, like Chimera1 freeze
  • wait - same as Chimera1 wait
  • title - static annotation like Chimera1 2dlabels, with some of the same options
  • perframe - similar to Chimera1 perframe
  • save
  • material
  • camera
  • oculus
  • pdbimages - batch-render images of molecular assemblies defined in mmCIF files (see examples)
