The ChimeraX Menu

Although most functionality is currently in commands, the ChimeraX menu includes basic file operations and undo, choices for selection, and access to several tools. See also: the ChimeraX window, context menus, Basic Actions, the Command Line

  • File
    • Open... a subset of open command functionality
    • Save... a subset of save command functionality
    • Close Session – same as the command close session

  • Edit – limited undo/redo
    • Undo
    • Redo

  • Select – what or how to select
    • Chains – chains in currently open structures
    • Chemistry
      • element – elements by atomic symbol
      • functional group
    • Menu mode: mode selection
      • replace (initial default mode)
      • add to (+)
      • subtract from (–)
      • intersect with (∩)

  • Tools – tools grouped by category

  • Presets – predefined combinations of display settings; the current set of presets is based directly on automatic styling and is likely to change.

    • Original Look - initial appearance from automatic styling, which depends on the number of chains in an atomic structure (however, certain aspects of appearance like silhouettes and cartoon style may differ if they have been changed since the structure was opened)
    • Cartoon - cartoons plus atomic detail near residues classified as ligand and ions, simple lighting, etc. (as for automatic styling of structures with 1-4 chains)
    • Space-Filling (chain colors) - all atoms as spheres except sticks for ligand and balls for solvent, chain colors, shadows, etc. (as for automatic styling of structures with 5-249 chains)
    • Space-Filling (single color) - all atoms as spheres, ambient-only lighting, etc. (as for automatic styling of structures with ≥250 chains)
    • Stick - all atoms as sticks, simple lighting, etc. (as for automatic styling of structures with 0 chains)

    More render-intensive presets such as Cartoon may exhibit slow performance on very large structures.
