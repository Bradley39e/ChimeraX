The ChimeraX Menu

Although most functionality is currently in commands, the ChimeraX menu includes basic file operations and undo, choices for selection, and access to several tools. See also: the ChimeraX window, context menus, Basic Actions, the Command Line

← File
  • Open... a subset of open command functionality
  • Open Folder... open all files of the specified format (currently DICOM only) in the specified directory and its subdirectories
  • Save... a subset of save command functionality
  • Close Session – same as the command close session
← Edit – limited undo/redo
  • Undo
  • Redo
← Select – what or how to select; see also the select command
  • Chains – chains in currently open structures
  • Chemistry
    • Element – elements by atomic symbol
    • Functional Group
    • IDATM Type – atom type
  • Residues – residues in currently open structures, sorted into nonstandard residues, standard nucleic acids, and standard amino acids
  • Structure
    • Backbone - biopolymer backbone atoms: peptide N, CA, C, O, nucleic acid phosphoribosyl backbone, and any directly attached hydrogens
    • Ions - ions
    • Ligand - ligand
    • Main - main
    • Nucleic Acid - nucleosides, nucleotides, RNA, and DNA
    • Protein - amino acids, peptides, and proteins
    • Ribose - nucleic-acid backbone ribose and doxyribose moieties
    • Secondary Structure - protein/peptide secondary structure types:
      • Coil - amino acid residues not in helix or strand
      • Helix - amino acid residues in helices
      • Strand - amino acid residues in strands
    • Sidechain + Connector - amino acid sidechain + CA (for connectivity to cartoon), nucleic acid base + ribose, and any directly attached hydrogens
    • Sidechain Only - amino acid sidechain, nucleic acid base, and any directly attached hydrogens
    • Solvent - solvent
  • Sequence... find and select a specified amino acid or nucleotide sequence or sequence pattern
  • Zone... select atoms and bonds (or whole residues, if Apply criteria to whole residues is checked) within and/or beyond a specified distance from the currently selected atoms. If both the within (<) and beyond (>) options are checked, both criteria must be met; for example, the atoms or residues must fall within 12 Å of but farther away than 8 Å from the current selection. OK performs the selection and closes the dialog, whereas Apply performs the selection without closing the dialog. See also: zone atom specification
  • Clear - deselect any existing selection
  • Menu Mode: mode
    • Replace (initial default mode)
    • Add (+)
    • Subtract (–)
    • Intersect (∩)
← Tools – tools grouped by category

← Presets – predefined combinations of display settings; the current set of presets is based directly on automatic styling and is likely to change. Render-intensive presets such as Cartoon may exhibit slow performance on very large structures.
  • Original Look - initial appearance from automatic styling, which depends on the number of chains in an atomic structure (however, certain aspects of appearance like silhouettes and cartoon style may differ if they have been changed since the structure was opened)
  • Cartoon - cartoons plus atomic detail near residues classified as ligand and ions, simple lighting, etc. (as for automatic styling of structures with 1-4 chains)
  • Space-Filling (chain colors) - all atoms as spheres except sticks for ligand and balls for solvent, chain colors, shadows, etc. (as for automatic styling of structures with 5-249 chains)
  • Space-Filling (single color) - all atoms as spheres, ambient-only lighting, etc. (as for automatic styling of structures with ≥250 chains)
  • Stick - all atoms as sticks, simple lighting, etc. (as for automatic styling of structures with 0 chains)
