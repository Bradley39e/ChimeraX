ChimeraX Window

ChimeraX has an all-in-one window interface in which panels can be shown, hidden, and repositioned. Panels can also be detached (undocked) from the main ChimeraX window and subsequently reinserted (docked). The overall window also contains the graphics area, a title bar across the top, and a status line across the bottom. See also: windowsize

A panel may include clickable icons (as in the Graphics Toolbar), tables, plots, histograms, and other 2D representations with interactive elements, or HTML with links and images (as in the Log and Help Viewer).

Panels are often tool interfaces, but may also appear as the result of a command (for example, contacts or help) or opening a certain type of data. A tool can be started or shown using the Tools menu and/or the toolshed command.

A panel can also be:

  • hidden by clicking the “x” button on its title bar (if the panel is docked) or by unchecking its name in the menu resulting from right-clicking any docked panel's title bar
  • shown (if active but hidden) by checking its name in the menu resulting from right-clicking any docked panel's title bar; some tools remain active when hidden, others do not
  • undocked from the main ChimeraX window by clicking the “rectangles” button on its title bar or by dragging or double-clicking its title bar
  • redocked by dragging or double-clicking its title bar, with the dragged location controlling where and how the panel will be inserted; multiple panels can be tabbed with each other so that they occupy the same space in the window, with only one shown at a time

Icon toolbars and the Command Line lack their own title bars, but the other methods of control are still available. The overall window or any undocked panel can be resized by dragging any corner and moved by dragging its title bar. The boundaries between docked panels or between panel(s) and the graphics area can be dragged to expand one side and shrink the other while keeping the overall window dimensions the same. Finally, the overall window can be toggled to hide/show the current panels by clicking the “nested squares” button near its bottom right corner. This allows expanding the graphics area with a single click without losing the overall window configuration. Icon toolbars and the Command Line are excluded from this toggling.

ChimeraX Browser and Help Viewer

The Help Viewer is a built-in HTML browser. ChimeraX help pages are written in HTML and can be shown in the Help Viewer by using the Help menu, the help command, tool context menus, and/or links in the Log. The Help Viewer opens as a separate window, but can be docked to and undocked from the main window like other panels, as described above. The name listed for the Help Viewer in panel title-bar menus will be the title of the HTML page it is showing.

The built-in browser provides a simple mechanism to execute ChimeraX commands by clicking special links, for example:

open 1gcn
– encoded as –
<a href="cxcmd:open 1gcn">open 1gcn</a>

The execution links will only work in the ChimeraX browser, not other web browsers. The Quick Start Guide includes many more examples of executable ChimeraX command links.

Arbitrary web content and HTML files (with or without such special links) can be shown in the ChimeraX browser with a command:

help URL
– or –
open local-file.html
Besides the Help Viewer, other panels with HTML content such as the Log are also displayed appropriately.
