ChimeraX Window

ChimeraX has an all-in-one window interface in which panels can be shown, hidden, and repositioned. The overall window also contains the graphics area, a title bar across the top, and a status line across the bottom.

A panel may include clickable icons (as in the Graphics Toolbar), tables, plots, histograms, and other 2D representations with interactive elements, or HTML with links and images (as in the Log). Panels are generally tool interfaces, but may also appear as the result of a command (for example, contacts). Tools can be called from the Tools menu or with the toolshed command. See also: windowsize

A panel can also be:

  • hidden by clicking the “x” button on its title bar or by unchecking its name in the menu resulting from right-clicking any panel title bar
  • shown (if started but hidden) by checking its name in the menu resulting from right-clicking any panel title bar
  • undocked so that it is separate from the main ChimeraX window by clicking the “rectangles” button on its title bar or by dragging or double-clicking its title bar
  • redocked by dragging or double-clicking its title bar, with the dragged location controlling where and how the panel will be inserted; multiple panels can be tabbed with each other so that they occupy the same space in the window, with only one shown at a time

Icon toolbars and the Command Line lack their own title bars, but the other methods of control are still available. The overall window or any undocked panel can be resized by dragging any corner and moved by dragging its title bar. The boundaries between docked panels or between panel(s) and the graphics area can be dragged to expand one side and shrink the other while keeping the overall window dimensions the same. Finally, the overall window can be toggled to hide/show the current panels by clicking the “nested squares” button near its bottom right corner. This allows giving more room to the graphics display in a single click without losing the overall window configuration. Icon toolbars and the Command Line are excluded from this toggling.

ChimeraX's built-in HTML browser is automatically used to show help pages. It also provides a simple mechanism to execute ChimeraX commands by clicking links, as in the Quick Start Guide; users can create their own HTML files, with or without such special links, and open them in ChimeraX with the help command.
