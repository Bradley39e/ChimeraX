Command: vr

Usage:
vr ( on | off )

Usage:
vr  roomPosition ( report | matrix )

The vr command enables a virtual reality mode for HTC Vive and Oculus Rift systems. The mode uses SteamVR, which must be installed separately by the user and started before the mode is enabled. In addition, Oculus Rift users should start the Oculus runtime before starting SteamVR. Synonyms for on: true, True, 1. Synonyms for off: false, False, 0.

The transformation between room coordinates (in meters, with origin at room center) and scene coordinates (typically in Å) can be specified or reported with vr roomPosition. The transformation matrix is given as 12 numbers separated by commas only, corresponding to a 3x3 matrix for rotation and scaling, with a translation vector in the fourth column. Ordering is row-by-row, such that the translation vector is given as the fourth, eighth, and twelfth numbers.

For details and related issues, see ChimeraX virtual reality. See also: camera, making movies
