Command: transparency

Usage:
transparency  atom-spec  percent  [ target  string ]

The transparency command sets the transparency of atomic and molecular representations to the specified percent, where 0% is completely opaque and 100% is completely transparent, without otherwise changing colors. See also: color, colordef, surface, volume

The absence of an atom or model specifier implies “all.” Keywords can be truncated to unique strings, and their case does not matter.

The target option allows acting on only a subset of the possible displays of a given atom, residue, etc. The string can be any combination of the following letters, without commas or spaces:

  • a - atoms/bonds (shown for bonds in halfbond mode)
  • b - bonds-only (shown for bonds not in halfbond mode)
  • c - cartoons (divisible to per-residue segments)
  • s - surfaces (molecular surfaces divisible to per-atom patches)
  • p - pseudobonds-only (shown for pseudobonds not in halfbond mode)
  • A - all of the above (equivalent to abcsp)
The default target is s, molecular surfaces only.
