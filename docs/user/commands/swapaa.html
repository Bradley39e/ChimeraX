Command: swapaa

Usage:
swapaa  residue-spec  new-type

The swapaa command performs a simple virtual mutation by replacing one type of amino acid with another. The new-type can be the three-letter code for any of the 20 standard amino acids (with or without capitalization), and one or more protein residues to change to that type can be specified in a single command.

The command will be improved to use a rotamer library, but currently sidechain replacement is done by matching a single conformation of the new residue type onto the backbone atoms of the existing residue. The backbone atoms are not replaced. See also: the swapaa mouse mode
