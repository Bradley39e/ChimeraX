Command: zonesel

Usage:
zonesel  ref-spec  cutoff  [ other-spec ] [ extend ]

The zonesel command selects atoms and/or surfaces in other-spec that are within a  cutoff  distance of the atoms and/or surfaces in ref-spec. A blank (omitted) other-spec indicates “all.” Surface models can be specified by model number or as a selection (details...). Although molecular surfaces can be included in the reference items (ref-spec), their membership in the resulting zone is not evaluated directly, because patches in a solvent-excluded surface are only selected if the corresponding atoms are selected. The extend keyword indicates that the reference items should also be selected. See also: select, sop zone

Examples:

zonesel ligand 4.5 protein
zonesel #1.2 8
zonesel sel 5 extend
