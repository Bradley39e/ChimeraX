Command: getcrd

Usage:
getcrd  atom-spec [ scene true | false ]

The getcrd command reports the coordinates of the specified atoms in the Log. The scene option specifies whether to include any rotations and translations of atomic models relative to the scene (default true). Otherwise, the untransformed coordinates will be reported. If only global rotations and transformations have been applied, this option makes no difference. See also: save
