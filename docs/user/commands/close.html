Command: close

Usage:
close model-spec

The command close removes the specified model(s) from Chimera2, including any submodels at lower levels of the hierarchy. For example, close #3 will close any models #3.N, #3.N.N, etc. Submodels may arise when molecular surfaces are generated or an ensemble (such as from NMR) is opened. The current model numbers can be shown in the Log using the list command. As per the usual convention, a blank specification indicates “all.” See also: open, quit

Examples:

close
close #1
close #2,4-6
close #3.2
close #3.2-10
