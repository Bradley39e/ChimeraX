Command: cartoon, ribbon

Usage:
( cartoon | ribbon )  atom-spec  [ adjust  factor ] [ hideBackbone true | false ] [ showSpine true | false ]

The cartoon or ribbon command shows a simplified backbone representation of the specified biopolymer residues, with additional settings for how closely the cartoon path should follow the backbone atoms and whether these atoms can be shown at the same time as the cartoon. The values of these settings can be different for different residues.

Commands hide and show should be used to undisplay and redisplay cartoons without modifying their associated parameters, whereas using cartoon again will reset any parameters not given explicitly in the command.

By default, showing the cartoon for a residue suppresses the display of its backbone atoms. Simultaneous display can be enabled with hideBackbone false; otherwise, the backbone atoms hidden by cartoons are N, C, and O in amino acids and the phosphoribose moieties of nucleic acids.

The adjust factor sets the balance between smoothing the cartoon path to ideality for residues in a peptide helix or strand (1.0) versus following the exact positions of the CA atoms (0.0). Each residue can have a different value. The default factor is 0.0 for amino acids in helices and 0.7 for those in strands. Smoothing is not applied to nucleic acids or amino acids in coil segments. The path-guiding atom is CA in amino acids and C5' in nucleic acids.

The showSpine option (default false) clarifies cartoon orientation by showing a series of bars across its width, somewhat resembling the treads of a tank (perhaps mainly for novelty purposes).
