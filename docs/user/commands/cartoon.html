Command: cartoon, ribbon

Usage:
( cartoon | ribbon )  atom-spec  [ adjust  default | factor ] [ hideBackbone true | false ]

Usage:
( ~cartoon | ~ribbon )  atom-spec 

The cartoon or ribbon command shows a simplified backbone representation of the specified biopolymer residues, with additional settings for path smoothing and whether a residue's backbone atoms can be shown at the same time as its cartoon segment. Settings can be different for different residues. Changes in settings apply only to the atomic models already present, not subsequently opened models.

The command ~cartoon or ~ribbon hides cartoon for the specified residues. The commands hide and show can also be used to control cartoon display.

The adjust factor sets the balance between smoothing the cartoon path for peptide helix or strand residues to ideality (1.0) versus following the exact positions of the α-carbons (0.0). Each residue can have a different value of factor, and lower values are recommended where sidechains are displayed, to preclude showing tethers. Specifying default instead of a numerical value restores the default of 0.0 for amino acid residues in helices and 0.7 for those in strands. Smoothing is not applied to peptide coil or to nucleic acids. The path-guiding atom in nucleic acids is C5'.

By default, showing the cartoon for a residue suppresses the display of its backbone atoms. Simultaneous display can be enabled with hideBackbone false; otherwise, the backbone atoms hidden by cartoons are N, C, and O in amino acids and the phosphate moieties including O5' in nucleic acids.

Usage:
( cartoon | ribbon ) tether  model-spec  [ shape cone | steeple | cylinder ] [ sides  N ] [ scale  scale ] [ opacity  opacity ]

A tether is drawn between a displayed α-carbon (CA atom) and the corresponding cartoon segment where they are offset (factor > 0). The command cartoon tether controls tether appearance. Tether parameters apply to an entire atomic model and cannot be different for different residues in the same model.

The tether shape can be:

  • cone (default) - cone with base at the α-carbon, point at the cartoon
  • steeple - cone with base at the cartoon, point at the α-carbon
  • cylinder

The sides option controls the number of planar facets used to draw a tether (range 3-10, default 4).

The scale option sets tether base radius relative to the display radius of the corresponding α-carbon (range 0.0-1.0, default 1.0).

The tether opacity value (1 – transparency) can range from 0.0 (completely transparent) to 1.0 (completely opaque) and defaults to 0.5. The tether color tracks the color of the corresponding α-carbon.
