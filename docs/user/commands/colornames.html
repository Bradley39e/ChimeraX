Command-Line Color Specification

Colors can be specified in commands with:

  • CSS color names, optionally with spaces between words
  • names defined previously with the colordef command
  • CSS hex format (hexadecimal)
  • CSS format for colorspaces: rgb (red, green, blue), rgba (red, green, blue, opacity), hsl (hue, saturation, lightness), hsla (hue, saturation, lightness, opacity), gray (grayscale)

The following all specify the same color:

red
#f00
#0xff0000
rgb(255, 0, 0)
rgb(100%, 0, 0)
rgba(100%, 0, 0, 1)
rgba(100%, 0, 0, 100%)
hsl(0, 100%, 50%)
The following specify the same shade of gray:
gray(128)
gray(50%)
