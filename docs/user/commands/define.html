Command: define

Usage:
define centroid  atom-spec [ massWeighting  true | false ] [ color  color-spec ] [ radius  r ] [ name  name ]

Define the centroid of the specified atoms with or without mass-weighting (default without). The centroid is added as a marker with the designated color (default the most prevalent color among the specified atoms, if at least 30% share that color, otherwise #909090
), radius (default 2.0 Å), and model name (default centroid). The related command measure center can also calculate center of mass for density maps. See also: measurements
