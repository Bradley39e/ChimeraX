Command: rename

Usage:
rename  model-spec  new-name

The command rename changes the name of the specified model(s) to new-name. If new-name includes spaces, it should be enclosed in quotes. The Model Panel will not show the change until its display is updated by some other action such as opening or closing a model or clicking a checkbox. See also: open

Example:

rename #1 mutant4
