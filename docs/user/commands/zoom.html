Command: zoom

Usage:
zoom  factor [ frames ]

The zoom command adjusts the apparent size of viewed items by factor at each of a specified number of image update frames (default 1 frame). The factor must be positive:

  • values > 1 enlarge the view
  • values < 1 shrink the view
Zooming moves the camera toward or away from the scene. It does not change model coordinates. See also: view, mousemode
