Command: colordef

Usage:
colordef  name [ color-spec ]

The colordef command defines a new color name or redefines an existing one. See also: color, transparency

The name is assigned as the name of the specified color. Color names must begin with an alphabetical letter and may contain only lowercase letters, numbers, hyphens, underscores, and spaces. If name has embedded spaces, it must be enclosed in quotation marks.

Examples:
colordef tpurple rgba(.6,.0,.8,.5)
colordef tgreen rgba(0%, 80%, 40%, 0.5)

The definition of a specific color can be shown in the Log with colordef name.
