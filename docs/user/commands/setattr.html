Command: setattr

Usage:
setattr  spec   level  attr_name   value  [ create  true | false ]

Setattr is a general, multipurpose attribute-setting command. It sets an attribute (existing or new) of the specified items at the indicated level to a single user-supplied value. The attribute may remain multi-valued, however, as only a subset of the items may have been specified. A new attribute will not be created unless create is true (default false).

Although setattr does not restrict which attributes it can affect, using it for certain attributes may not make sense. In some cases, another command already exists to perform the same task. For example, the following are equivalent:

setattr /a res ribbon_color orange
color /a orange targ r

The level can be any one of the following, with truncations allowed:

  • atoms
  • residues
  • chains
  • bonds
  • pseudobonds - individual pseudobonds
  • structures - atomic models
  • surfaces - surface models
  • groups - pseudobond models
  • models - models of all applicable types
