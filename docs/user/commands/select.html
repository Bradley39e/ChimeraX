Command: select

Usage:
select  spec  [ entity  chain-spec ]

Usage:
select ( add | subtract | intersect )  spec

Usage:
~select  spec

Usage:
select clear

Usage:
select zone  ref-spec  cutoff  [ other-spec ] [ extend  true | false ] [ residues  true | false ]

A selection specifies items (models or their parts) for subsequent operations and is shown with green outlines in the graphics window. The select command makes a new selection or modifies an existing selection. See also: mousemode, surface zone

If used without an add, subtract, or intersect keyword, the select command replaces any previous selection with a new selection containing the items in spec. The entity option restricts that selection to only the biopolymer chains with the same sequences (the same macromolecular entities) as the chains in chain-spec. Examples:

select #1 entity /a
select ent /d

Otherwise, select can be used to add, subtract, or intersect the specified items with the current selection, or to clear it by deselecting everything. The ~select command can be used to deselect specified items (like select subtract) or with a blank specification to deselect everything (like select clear).

The select zone command selects atoms and/or surfaces in other-spec that are within a  cutoff  distance of the atoms and/or surfaces in ref-spec. A blank (omitted) other-spec indicates “all.” Although molecular surfaces can be included in the reference items (ref-spec), their membership in the resulting zone is not evaluated directly because their selection is tied to that of the corresponding atoms. The extend option (default false) indicates whether the reference items should also be selected. The residues option (default false) indicates whether to select all atoms of a residue if any atom in the residue is within the zone. Examples:

select zone ligand 4.5 protein res t
sel zone #1.2 8
sel zone sel 5 extend t
