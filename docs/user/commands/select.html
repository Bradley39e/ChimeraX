Command: select

Usage:
select  atom-spec  [ sequence  chain-spec ]

Usage:
select ( add | subtract | intersect )  atom-spec

Usage:
~select  atom-spec

Usage:
select clear

The select command makes a new selection or modifies an existing selection. A selection:

  • is shown with green outlines in the graphics window
  • specifies items (models or their parts) for subsequent operations:
    • indicates scope of action of Molecule Display icons
    • can be specified as sel or selAtoms in other commands, for example: style sel stick
  • can be created
    • by Ctrl-clicking on items in the graphics window (default; see mousemode)
    • by using select or various other commands and tools
  • can be broadened with the up arrow key, narrowed with the down arrow key
    • selection cascade: atom ↔ residue ↔ all atoms with same chain ID ↔ atomic model

If used without an add, subtract, or intersect keyword, the select command replaces any previous selection with a new selection containing the items in atom-spec. The sequence option restricts that selection to only the biopolymer chains with the same sequences (the same molecular entities) as the chains in chain-spec. Examples:

select #1 sequence /a
select seq /d

Otherwise, select can be used to add, subtract, or intersect the specified items with the current selection, or to clear it by deselecting everything. The ~select command can be used to deselect specified items (like select subtract) or with a blank specification to deselect everything (like select clear).

Selecting a molecular surface automatically also selects the atoms used to define it. If the whole surface model is selected, all of the atoms used to define that surface are selected, even atoms that are completely buried. Typically all of the atoms in a biopolymer chain are used to define a surface, but other sets of atoms can be specified. If a surface patch is selected with the mouse, only the atom(s) directly contributing to that patch are selected.
