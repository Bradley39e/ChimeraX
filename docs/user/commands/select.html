Command: select

Usage:
select  atom-spec

Usage:
select ( add | subtract | intersect )  atom-spec

Usage:
select clear

The select command makes a new selection or modifies an existing selection. A selection:

  • is shown with green outlines in the main window
  • specifies items (models or their parts) for subsequent operations, namely:
    • indicates scope of action of Molecule Display icons
    • can be specified in subsequent commands with the word sel
  • can be made by clicking on items in the main window or by using various commands (including select) and tools
  • can be broadened with the up arrow key, narrowed with the down arrow key

If no keyword option is used, select creates a new selection containing the specified items, replacing any previous selection.

Otherwise, select can be used to add, subtract, or intersect the specified items with the current selection, or to clear it by deselecting everything.
