Command: cd

Usage:
cd [ directory ]

The command cd changes the current working directory as specified, or if nothing is specified, to the user's home directory. Substituting the word browse for the directory brings up a file browser window for choosing it interactively. Alternatively, dragging and dropping the icon for a file or directory into the Command Line Interface substitutes the corresponding pathname text. The working directory is reported in the Log. See also: pwd, open, save
