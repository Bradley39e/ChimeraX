Command: mousemode

Usage:
mousemode [ control ] [ alt ] [ command ] [ shift ] [ leftMode  function ] [ middleMode  function ] [ rightMode  function ] [ wheelMode  function ] [ pauseMode  function ]

The mousemode command assigns ChimeraX functions to the left, middle, and right mouse buttons, alone and in combination with the modifier keys control, alt, command, and shift. In addition, wheel refers to scrolling with the mouse wheel or touchpad, and pause refers to pausing the cursor over something in the graphics window without clicking any buttons.

Possible values of function are listed in the table below. If function contains spaces, it should be enclosed in quotation marks.

Multiple modifiers and multiple “buttons” (including wheel and pause) can be given in a single command, but all specified modifiers apply to all specified buttons. For example, two commands rather than one would be needed to assign functions to control right and alt wheel:

mousemode control right 'rotate and select'
mousemode alt wheel 'contour level'

In the case of a touchpad or single-clicker mouse, middle- and right-button emulation are normally provided by the alt and command keys, respectively. It may be best to leave alt left and command left unassigned if they will be needed for this purpose.

The mousemode command without arguments reports the current assignments in the Log.
