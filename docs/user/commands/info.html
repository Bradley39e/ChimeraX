Command: info

Usage:
info  model-spec

The command info shows information on the specified models in the Log. As per the usual convention, a blank specification indicates “all.” The type of information depends on the type of model, and includes the numbers of atoms, bonds, residues, and chains in an atomic model, the number of coordinate sets in a trajectory model, and the value range and current contour level(s) of a volume data model. See also: open, listinfo
