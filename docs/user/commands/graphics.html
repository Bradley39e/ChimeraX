Command: graphics

Usage:
graphics [ atomTriangles  Na ] [ bondTriangles  Nb ] [ ribbonDivisions  Nd ] [ ribbonSides  Ns ]

Usage:
graphics restart

Normally, the fineness of atom and bond triangulation is adjusted automatically depending on the display contents (e.g., the number of atoms displayed), within the ranges of 10-2000 triangles per atom and 24-160 per bond. Automatic adjustment can be tuned in either direction with set subdivision.

The graphics command allows instead using fixed numbers of triangles:

  • atomTriangles Na - triangles per atom (minimum 4, except that 0 specifies going back to automatic adjustment)
  • bondTriangles Nb - triangles per bond (minimum 12, except that 0 specifies going back to automatic adjustment)
  • ribbonDivisions Nd - divisions per residue lengthwise along the cartoon (initial default 20, minimum 2)
  • ribbonSides Ns - number of sides used to approximate an oval or round cartoon cross-section (initial default 12, minimum 4)

Without options, graphics reports the current numbers for atoms, bonds, and ribbon divisions in the Log. The number of ribbon sides and divisions can be reported (or changed) with cartoon style. See also: style

The command graphics restart reinitiates graphics drawing after an error.
