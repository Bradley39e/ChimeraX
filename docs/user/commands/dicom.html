Command: dicom

Usage:
dicom  view ( default | fourup | overunder | sidebyside ]

The command dicom view sets the layout of medical image display, where “medical image” refers to volume data opened from the DICOM, NIfTI (.nii), and/or NRRD (.nrrd) formats. See also: view, ChimeraX DICOM Reference, the Medical Image toolbar, ChimeraX for Medical Image Analysis

Choices of layout include:

  • default – a single view in the graphics window
  • fourup – four panes in the graphics window, arranged 2x2: axial slice (top left), 3D rendering (top right), coronal slice (bottom left), sagittal slice (bottom right)
  • overunder – 3D view across the top, the three orthogonal slice views arranged horizontally across the bottom
  • sidebyside – 3D view on the left, the three orthogonal slice views arranged vertically on the right

Each slice view has a slider for navigating through the data planes along the corresponding axis.
