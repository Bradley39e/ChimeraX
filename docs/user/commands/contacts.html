Command: contacts

Usage:
contacts  atom-spec  [ probeRadius  rad ] [ areaCutoff  area ]

The contacts command calculates the buried solvent-accessible surface area (SASA) for each pair of chains in the specified set of atoms, based on chain IDs, and generates a network diagram of the interchain contacts. Unspecified atoms are ignored, but solvent, ions, and ligand residues are not excluded automatically. The buried SASA is calculated as described for buriedarea, and the measured interface areas are reported in the Log. See also: sasa

Whereas the surface command shows solvent-excluded surfaces (SES) defined by the closest approach of the surface of a probe sphere rolled over the atomic VDW spheres, the solvent-accessible surface (SAS) is traced out by the center of the probe sphere and is not displayed. The probeRadius rad is the radius of this “rolling sphere” probe (default 1.4 Å, the value often used to approximate a water molecule). Interfaces with at least area buried (default 300 Å2) are counted as contacts and shown as lines in the diagram.

Chain-Chain Network Diagram

In the 2D network diagram created by the contacts command, the nodes (circles) represent chains, and the edges (lines) between nodes represent chain-chain interfaces. Node colors match the predominant colors of the chains when the diagram is created, but will toggle between that initial color when a chain is shown and gray when it is hidden. Node sizes are proportional to chain surface areas. Smaller interfaces (buried area less than half that of the largest interface in the structure) are indicated with dashed lines.

Clicking a node shows the atoms of the corresponding chain and any interacting chains; other nodes are grayed out and their atoms hidden. Shift-clicking a node selects the atoms of the corresponding chain. Similarly, clicking an edge shows the atoms of the two connected nodes and grays out other nodes and hides their atoms, while Shift-clicking an edge selects the atoms of the two nodes. Clicking the background of the diagram shows all of the chains. Only atoms included in the calculation are affected by these actions; other atoms are ignored.

Showing/hiding atoms by interacting with the diagram does not affect cartoon or surface displays.

The diagram can be moved as a whole with left mouse drag (initially clicking on the background, not on a node or edge) and scaled with Shift-left mouse drag. Scaling does not affect node sizes, only their spacing. The panel containing the diagram can be manipulated like other ChimeraX panels.

Context menus are accessed by right-clicking various parts of the chain-chain network diagram. Most of the menu entries act on only the atoms used in the buried-area calculations, as specified in the contacts command.

The background context menu includes:

  • Show all atoms - display all atoms used in the calculation (same as clicking the diagram background)
  • Select contact residues - select only the interface residues (with SASA buried in an interface) from all chains
  • Select all - select all atoms used in the calculation
  • Clear selection - clear the current selection
  • Explode - move chains apart to separate the interfaces
  • Unexplode - move atoms back together to restore the original structure
  • Layout matching structure - redraw diagram with orientation similar to the current 3D structure
  • Orient structure - rotate the structure to an orientation similar to the 2D diagram
  • Hide this tool - hide the panel (can be reshown)
The node context menu includes (indented entries the same as in the background context menu):

  • Show only [chain] - show the corresponding chain; hide others
  • Show [chain] and neighbors - show only the corresponding chain and its contacting chains (same as clicking the node)
  • Show contact residues - show the corresponding chain with its interface residues in light gray, plus only the relevant interface residues from its contacting chains (not their interface residues with other chains)
    • Show all atoms - display all atoms used in the calculation (same as clicking the diagram background)
  • Select [chain] - select the corresponding chain (same as Shift-clicking the node)
  • Select neighbors - select chains contacting the corresponding chain
  • Select contact residues - select the interface residues on both sides of all interfaces of the corresponding chain
    • Select all - select all atoms used in the calculation
    • Clear selection - clear the current selection
    • Explode - move chains apart to separate the interfaces
    • Unexplode - move atoms back together to restore the original structure
    • Layout matching structure - redraw diagram with orientation similar to the current 3D structure
    • Orient structure - rotate the structure to an orientation similar to the 2D diagram
    • Hide this tool - hide the panel (can be reshown)

The edge context menu includes (indented entries the same as in the background context menu):

  • Show only [chain1,chain2] - show the corresponding chains; hide others (same as clicking the edge)
  • Show [chain1] contact residues - show chain2 as spheres with its chain1-chain2 interface residues in light gray, plus only the chain1-chain2 interface residues from chain1; adjust view to interface, orient towards chain2
  • Show [chain2] contact residues - show chain1 as spheres with its chain1-chain2 interface residues in light gray, plus only the chain1-chain2 interface residues from chain2; adjust view to interface, orient towards chain1
    • Show all atoms - display all atoms used in the calculation (same as clicking the diagram background)
  • Residue plot - create a residue plot, essentially a 2D representation of what would be obtained in 3D with the menu item Show [chain2] contact residues (above): colored circles for individual chain1-chain2 interface residues superimposed on a gray silhouette of the remaining residues of chain1
  • Select [chain1,chain2] - select the corresponding chains (same as Shift-clicking the edge)
  • Select contact residues - select the chain1-chain2 interface residues of both chains
  • Select [chain1] contact residues - select the chain1-chain2 interface residues of chain1
  • Select [chain2] contact residues - select the chain1-chain2 interface residues of chain2
    • Select all - select all atoms used in the calculation
    • Clear selection - clear the current selection
    • Explode - move chains apart to separate the interfaces
    • Unexplode - move atoms back together to restore the original structure
    • Layout matching structure - redraw diagram with orientation similar to the current 3D structure
    • Orient structure - rotate the structure to an orientation similar to the 2D diagram
    • Hide this tool - hide the panel (can be reshown)

Interface Residue Plot

A plot of the chain-chain interface residues can be shown using the edge context menu in a chain-chain network diagram. The residue plot is essentially a 2D projection of what would be obtained in 3D with Show [chain2] contact residues in the chain1-chain2 edge context menu: colored nodes (circles) for individual chain1-chain2 interface residues superimposed on a gray silhouette of the remaining residues of chain1. The interface-residue nodes are colored to match the structure and labeled by residue number. Clicking a node selects the corresponding residue in the structure.

The plot can be moved as a whole with left mouse drag (initially clicking the background, not on a node) and scaled with Shift-left mouse drag. Scaling does not affect node sizes, only their spacing. The panel containing the plot can be manipulated like other ChimeraX panels.

The residue plot has its own context menus, with:

  • Show interface - in 3D, show the chain (gray in the plot) as spheres with its interface residues in light gray, plus only the interface residues from the other chain; adjust the view to match the plot (same as Show [chain2] contact residues in the chain1-chain2 edge context menu)
  • Select residue (in residue context menu) - select the corresponding residue in the structure (same as clicking the node)
  • Hide this tool - hide the panel (can be reshown)
