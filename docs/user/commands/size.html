Command: size

Usage:
size  atom-spec [ atomRadius  ra | default ] [ bondRadius  rb ]

The size command adjusts the radii of specified atoms and/or bonds:

  • atomic VDW radii, used directly for sphere display and scaled by a factor of 0.3 for ball display (default radii assigned by atom type)
  • bond radii for stick display (default 0.2 Å)
Atomic VDW radii are also used to calculate molecular surfaces and surface areas.

See also: style, surface, sasa, buriedarea, contacts

Examples:

size F atomRadius 1.5
size #1 atom default
size bond 0.4
