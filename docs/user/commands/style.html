Command: style

Usage:
style  spec  [ sphere | stick | ball ] [ atomRadius  ra | default ] [ stickRadius  rb ] [ ballScale  b ] [ dashes  N ]

The style command sets display style and/or size of atoms and bonds. The style can be one of the following:

  • sphere - space-filling with full atomic VDW radii (the default radii are assigned by atom type)
  • stick - sticks with rounded ends and stickRadius rb (initial default 0.2 Å)
  • ball - ball-and-stick with atomic VDW radii scaled by the ballScale factor (initial default 0.3) and bonds shown as sticks with stickRadius rb (initial default 0.2 Å)

Note that VDW radii are also used to calculate molecular surfaces, surface areas, and contacts based on buried surface areas. The ballScale parameter applies to an entire atomic model even if only part of the model is specified.

The style command can also set the number of dashes per pseudobond. Bonds and pseudobonds are considered as specified when both flanking atoms are specified, and all of the pseudobonds in a group will be affected in parallel. Pseudobonds can also be specified by pseudobond-group model number. Only whole numbers of dashes are accepted, with 0 indicating solid sticks. Currently, odd numbers are rounded down to give an even number of dashes.
