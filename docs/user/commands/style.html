Command: style

Usage:
style  atom-spec [ stick | ball | sphere ] [ dashes  N ]

The style command sets the display style of the specified atoms/bonds to one of the following:

  • stick - sticks with rounded ends
  • ball - ball-and-stick with VDW radii scaled by a factor of 0.3
  • sphere - space-filling with full VDW radii
In addition, style can set the number of dashes per pseudobond. Bonds and pseudobonds are considered as specified when both flanking atoms are specified, and all of the pseudobonds in a group will be affected in parallel. Only whole numbers are accepted, with 0 indicating solid sticks. Currently, odd numbers are rounded down to give an even number of dashes.

Stick radii and atomic VDW radii can be adjusted with the size command. See also: show, Molecule Display Toolbar
