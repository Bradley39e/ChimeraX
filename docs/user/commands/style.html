Command: style

Usage:
style  atom-spec [ stick | ball | sphere ]

The style command sets the display style of the specified atoms/bonds to one of the following:

  • stick - sticks with rounded ends
  • ball - ball-and-stick with scaled-down van der Waals (VDW) radii
  • sphere - space-filling with full VDW radii
As per the usual convention, a blank specification means “all.” See also: show
