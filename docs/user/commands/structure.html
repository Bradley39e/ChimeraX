Command: structure

Usage:
structure modify  atom   element   numBonds  [ geometry  ion | single | linear | trigonal | tetrahedral ] [ connectBack  true | false ] [ colorByElement  true | false ] [ name  name ] [ resName  resname ] [ resNewOnly  true | false ]

Subcommands of structure correspond to sections of the Build Structure tool.

The structure modify command changes the type of a single specified atom and automatically fills its valence with attached hydrogens. The hydrogens could be modified in turn to continue building outward. The element (element symbol) and numBonds (total number of bonds to the atom, including hydrogens) must also be given, for example:

structure modify sel O 2 geometry tetrahedral

See also: addh, delete, swapaa

Hydrogens will be added to the atom to generate the indicated total number of bonds. Pre-existing atoms are not removed. If the atom is already bonded to one (and only one) other atom, the bond will be adjusted to an approximate length depending on the elements involved. The positions of any other atoms already present will not be changed. New hydrogens will be placed to form the idealized bond angles for the specified geometry, or if the atom already has two or more substituents, to maximally avoid those substituents. The default geometry is the same as for the atom's current type.

Other options:

  • connectBack (default true) – if a newly added hydrogen would be very close to an existing atom in the same model, discard the hydrogen and form a bond to the existing atom instead
  • colorByElement (default true) – color the modified atom and any newly added hydrogens by element
  • name – new name for the modified atom; if not given, the atom name is kept the same
  • resName – new name for the residue containing the modified atom; if not given, the residue name is kept the same
  • resNewOnly (default false) – if a new residue name is given, whether to put only the new atom(s) in a new residue of that name instead of changing the name of the existing residue
