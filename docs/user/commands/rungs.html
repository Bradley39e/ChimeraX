Command: rungs

Usage:
rungs  atom-spec  [ radius r ] [ color color-spec ] [ halfbond  true | false ] [ hideAtoms  true | false ] [ hideHbonds  true | false ] [ showRibbon  true | false ]

Usage:
rungs hide  atom-spec

The rungs command shows a simple broken-ladder representation of the nucleic acid residues containing the specified atoms, or if nothing is specified, all nucleic acid residues. The rung representation for a nucleic acid residue is a cylinder from the ribose C3' to N1 (if purine) or N3 (if pyrimidine), implemented as a pseudobond. The command rungs hide undisplays any rungs of the specified residues. See also: show, color, cartoon, struts

The initial default rung radius r is 0.5 Å, and the initial default color is to match the cartoon of each residue. If halfbond is true (initial default false), the color setting is not used; instead, the two halves of each rung will be colored to match the endpoint atoms, and will track any subsequent color changes of those atoms. Once created, rung pseudobonds retain their existing radius and color settings unless those settings are changed explicitly.

The hideAtoms option indicates hiding the atoms/bonds of the specified residues (default true), not including the rung endpoint atoms, which must remain shown for the rungs themselves to be shown. The hideHbonds option (default true) indicates hiding any H-bond pseudobonds. The showRibbon option (default true) indicates whether to show the ribbon (cartoon) segments of the residues. Setting a hiding option to false will not show already hidden items, and setting showRibbon to false will not hide any cartoon segments that are already shown.

The following commands can be used to show nucleic acid atoms and their pre-existing H-bonds again after rungs have been hidden:

show nucleic
show hbonds

The second command may not be needed if the H-bonds were only invisible because their endpoint atoms were hidden.
