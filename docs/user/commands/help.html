Command: help

Usage:
help  command

Usage:
help  URL

The command help shows the help page for the specified command, if available, in the ChimeraX internal browser. If the command exists but lacks a help page, its usage information is shown instead. Alternatively, a web page or local file can be specified by URL for viewing in the internal browser. See also: open

Examples:

help set
help vop gaussian
help http://www.rbvi.ucsf.edu/chimerax/features.html
