Command: runscript

Usage:
run  command [ log  true | false ] [ downgradeErrors  true | false ]

The run command runs another ChimeraX command, which should be enclosed in quotation marks if it (along with its arguments) includes any spaces. See also: runscript

The log option specifies whether to send messages to the Log (default true). The downgradeErrors option indicates whether to downgrade error messages, which would raise a dialog and halt script execution, to warnings that would simply go to the Log (default false).
