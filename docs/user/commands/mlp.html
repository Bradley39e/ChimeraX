Command: mlp

Usage:
mlp  model-spec  [ dx  pathname ] [ spacing s ] [ method dubost | fauchere | brasseur | buckingham [ nexp n ] | type5 ]

The command mlp generates molecular lipophilicity potential (MLP) maps for protein structures using the program pyMLP:

Molecular structures: Perception, autocorrelation descriptor and SAR studies. System of atomic contributions for the calculation of the n-octanol/water partition coefficients. Broto P, Moreau G, Vandycke C. Eur J Med Chem. 1984;19(1):71-78.

MLPP: A program for the calculation of molecular lipophilicity potential in proteins. Laguerre M, Saux M, Dubost JP, Carpy A. Pharm. Sci. 1997;3(5-6):217-22.

The pyMLP code includes lookup values for nonhydrogen atoms in the 20 standard amino acids plus HYP (hydroxyproline) and PCA (pyroglutamic acid). Any other residue types are ignored. Positive potentials correspond to more lipophilic (more hydrophobic) areas, negative to less lipophilic (more hydrophilic) areas. Warning: execution of this command is rather slow.

See also: open, volume

The output potential map will be opened automatically. The dx option specifies a name and location (pathname) for the output map in DX format; otherwise, a temporary name and location will be used.

The spacing s of the map grid in Å can be specified in the range 0.1–10.0 (default 1.0).

The method indicates how atomic values propagate through space, with multiplicative factors based on the distance d from the atom. The potential at each grid point is a sum over the atomic contributions.

  • dubost: (1+d)–1
  • fauchere (default): e–d
  • brasseur: e–d/3.1
  • buckingham: d –n
    ... n given with the nexp option (default 3.0)
  • type5: e–d ½
