Command: crossfade

Usage:
crossfade [ frames ]

The crossfade command fades from the current display to the next change in display (from the current frame to what would have been the next drawn frame) over a specified number of image display frames, default 30. The subsequent command(s) can be anything that changes the display. However, crossfade simply interpolates each pixel linearly; gradually changing the orientation or position of some object instead requires using frame arguments in the commands performing those changes (move, turn, view, etc.). See also: making movies

Examples:

crossfade; hide ligand
crossfade; show ligand
crossfade 60; style ligand sphere
crossfade 60; set bg light blue
crossfade 100; light full
