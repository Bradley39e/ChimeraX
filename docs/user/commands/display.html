Command: display

Usage:
display  spec [ level ]

Usage:
~display  spec [ level ]

Display and ~display show and hide the specified atoms, bonds, cartoons, and entire models. Following the usual convention, a blank specification means “all.” See the command surface for showing and hiding molecular surfaces. See also: style, cartoon, volume

The level can be:

  • atoms - specified atoms
  • bonds - bonds with both end atoms specified; includes end atoms for display (since they are required to display the bond, see hierarchy) but not for ~display
  • pseudobonds or pbonds - pseudobonds with both end atoms specified; includes end atoms for display (since they are required to display the pseudobond, see hierarchy) but not for ~display
  • cartoons or ribbons - cartoon (ribbon) segments of the specified residues
  • models - entire models, without affecting the display status of their subparts (for example, which atoms are shown; see hierarchy)
The level argument is optional and can be truncated. The default level is atoms when spec includes atoms, but models when only nonatomic (e.g., volume) models are specified. A blank specification includes atoms when atomic models are present.
