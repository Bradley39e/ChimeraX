UCSF ChimeraX Advantages

ChimeraX provides several advantages relative to Chimera, and the list will continue to grow. A small sample is given here:

  • High-performance manipulation and rendering of large numbers of atoms

  • Fast mmCIF reader

  • Fast and robust solvent-excluded surfaces with clean patch boundaries

  • Ambient occlusion lighting mode

  • Icons to change display styles, lighting modes, or mouse-button assignments with a single click

  • All-in-one window in which tool interfaces are panels that can be shown, hidden, and repositioned

  • Each tool panel may display HTML with links and images, plots, or network diagrams

  • Built-in HTML browser with simple mechanism to execute ChimeraX commands by clicking links (handy for demonstrations)

  • Tool Shed manager allowing tools to be installed or updated separately, with dependencies handled correctly

  • Virtual reality camera mode for Oculus Rift goggles
