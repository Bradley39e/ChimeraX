ChimeraX Preferences (Settings)

The ChimeraX Settings dialog is listed in a standard platform-dependent location in the menu (for example, UCSF ChimeraX... Preferences on a Mac), and contains sections:

Background
Clipping
Distances
Labels
Sessions
Toolshed

Save saves the current settings, Reset replaces the current settings with the initial (“factory”) defaults, and Restore restores values that were saved previously. The Buttons below... option indicates whether these buttons should apply only to the currently shown section or to all sections of the Preferences dialog.

Several other preferences do not appear in this dialog, but in separate, tool-specific interfaces.

ChimeraX preferences are stored as multiple files in a standard platform-dependent location (for example, under ~/Library/Application Support/ChimeraX/ on a Mac). The location can be reported with command:

info path user unversioned config
Preferences files are named according to the modules to which they apply, and are not created until nondefault settings are saved.

← Background

  • Background color - a color well that can be clicked to choose the color interactively (initial default
    ). See also: set bgColor

← Clipping

  • Surface caps - whether to cap surfaces initially when they are sliced by clipping. See also: surface cap

← Distances

Settings for distance-monitor pseudobonds are saved instantly (without clicking Save). See also: distance, mousemode, selection context menus

  • Color - a color well that can be clicked to choose the color interactively (initial default
    )
  • Decimal places - distance-label places after the decimal (initial default 3)
  • Number of dashes (initial default 9)
  • Radius (initial default 0.1 Å)
  • Show angstrom symbol (Å) - whether to include the Å symbol in the distance label (initial default true)

← Labels

  • Atomspec display style - format/type of identifying information shown by pausing the cursor over an object in the main window (see the identify object mouse mode)
    • command line - command-line specifier (for example, /B:21@OD1)
    • serial number - atom serial number (for example, 126)
    • simple (initial default) - chain ID (preceded by slash), residue name, residue number, atom name (for example, /B ASN 21 OD1)

    Model number will be included when multiple models are present.

← Sessions

  • Resize window on session restore - whether to restore the window size of a session along with its contents; if this option is off (initial default), the graphics window will not resize when a session is restored. See also: windowsize

← Toolshed

The Toolshed is the web repository of ChimeraX bundles.

  • Toolshed update interval - how often to check for updates
    • every day
    • every week (default)
    • every month

Other Preferences

Some preferences do not appear in the main ChimeraX Settings dialog, but can be adjusted and saved by other means:

  • Individual tool preferences: whether to autostart, whether windows are dockable, and other settings accessible through context menus
