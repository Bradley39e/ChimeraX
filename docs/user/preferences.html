ChimeraX Preferences

A user's ChimeraX preferences are stored along with the command history and file history as multiple files in a standard platform-dependent location (for example, under ~/Library/Application Support/ChimeraX/ on a Mac). The location can be reported with command:

info path user all config
Preferences files are named according to the modules to which they apply, and are not created until nondefault settings are saved.

The main Preferences dialog is listed in a standard platform-dependent location in the menu (for example, UCSF ChimeraX... Preferences on a Mac), and contains sections:

Background
Labels

Save saves the current settings, Reset replaces the current settings with the initial (“factory”) defaults, and Restore replaces the current settings with those previously saved; if the Buttons affect all categories option is checked, these buttons apply to all sections of the Preferences dialog, not just the section currently shown.

Background

  • Background color - a color well that can be clicked to choose the color interactively (initial default
    )

Labels

  • Atomspec display style - format/type of identifying information shown by pausing the cursor over an object in the main window (see the identify object mouse mode)
    • simple (initial default) - chain ID (preceded by slash), residue name, residue number, atom name (for example, /B ASN 21 OD1)
    • command line - command-line specifier (for example, /B:21@OD1)
    • serial number - atom serial number (for example, 126)

    Model number will be included when multiple models are present.

Other Preferences

Some preferences do not apear in the dialog, but can be adjusted and saved by other means:

  • Individual tool preferences: whether to autostart, whether windows are dockable
  • Distance-label formatting, see distance format
