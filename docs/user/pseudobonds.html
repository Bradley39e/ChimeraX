Pseudobonds

A pseudobond is drawn between two atoms to signify a connection other than a covalent bond, such as a hydrogen bond, metal coordination bond, or missing peptide segment. A pseudobond model is a named set of pseudobonds that can be treated collectively. Pseudobonds can be defined arbitrarily by creating and opening a pseudobond file. See also: hbonds, rungs, struts

Pseudobonds are only displayed when both of their endpoint atoms are displayed. However, a backbone atom is considered as displayed when its cartoon is suppressing the backbone atoms. Thus, a pseudobond (which goes the atom position) may appear to be floating in space. Floating pseudobonds can be hidden with the command cartoon suppress false, with the side effect of showing any backbone atoms that were already displayed but suppressed. See also: display hierarchy

Commands that act on pseudobonds include color, style, show, and crosslinks. For example, an interactive histogram of pseudobond lengths can be plotted with crosslinks histogram. The command-line specifiers hbonds and pbonds refer to hydrogen-bond pseudobonds and all pseudobonds, respectively, as well as their endpoint atoms. Certain commands (color, show) may require using the target p option to indicate acting on pseudobonds and not their flanking atoms.
