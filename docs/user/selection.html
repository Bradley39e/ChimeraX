Selection

A selection is a set of items (atoms, residues, models) chosen by the user:

  • shown with green outlines in the graphics window
  • marks the items for subsequent operations:
    • can be specified as sel in many commands, for example: style sel stick
    • indicates scope of action of Molecule Display icons and Density Maps icons
  • can be created:
    • with the mouse in the graphics window, by default (see mousemode):
      • Ctrl-click to start selection (replace)
      • Shift-Ctrl-click to add/subtract (toggle)
    • with the select command
    • with various other commands and tools
  • can be broadened with the up arrow key, narrowed with the down arrow key, within selection cascade:
    • atom ↔ residue ↔ all atoms with same chain ID ↔ atomic model
