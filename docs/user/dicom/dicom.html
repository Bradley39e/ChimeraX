DICOM Files

DICOM (Digital Imaging and Communications in Medicine) refers to a broad and complex set of standards for medical images and related information. This input format is indicated by the filename suffix .dcm or the format dicom option in the open command. See also: DICOM Quick Reference, ChimeraX for Medical Image Analysis

All of the DICOM files in a directory and its subdirectories can be opened in a single operation if the format is specified explicitly:

Menu: File... Open Folder (dialog shows Format: DICOM image), browse to location, choose directory, click Open
– or –
Command: open  directory-pathname  format dicom

...where directory-pathname may contain * as wildcard to lessen typing, and should be enclosed in quotation marks if it includes spaces. Each image stack that is found will be opened as a separate model.

Alternatively, multiple DICOM files comprising a single 3D image stack can be opened as follows:

Menu: File... Open, browse to location, click first file, Shift-click last file to select the entire set, click Open
– or –
Command: open  pathname-glob

...where pathname-glob specifies the name and location of multiple files, typically using * as wildcard and enclosed in quotes if it contains spaces.

The open command verbose option shows detailed debugging information for DICOM input.

Some DICOM files contain time series that can be played with the time slider shown automatically by ChimeraX or with the vseries command.

Some DICOM files contain SEG objects (segmentations as binary maps), and these are automatically associated with the corresponding image series to synchronize plane display. They can also be displayed as isosurfaces, for example: volume #1.1.1.1.5 show style surf level 1 region all

DICOM Model Hierarchy

Models read from DICOM files are arranged in a hierarchy, as shown in the Model Panel:

  1. directory
  2. patient ID
  3. study date
  4. series description and number, if available, or body part and modality
  5. segmentations or other image series-dependent annotations (for example, segmentations of a particular CT scan)
