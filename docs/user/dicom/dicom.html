DICOM Files

DICOM (Digital Imaging and Communications in Medicine) refers to a broad and complex set of standards for medical images and related information. This input format is indicated by the filename suffix .dcm or the format dicom option in the open command. See also: DICOM Quick Reference

Generally multiple DICOM files comprise a single 3D image. These can be opened as follows:

Menu: File... Open, browse to location, click first file, Shift-click last file to select the entire set, click Open
– or –
Command: open  pathname-glob

...where pathname-glob specifies the name and location of multiple files, typically using * as wildcard and enclosed in quotes if it contains spaces.

It is also possible to open all of the DICOM files in a directory and its subdirectories in a single operation if the format is specified explicitly:

Menu: File... Open Folder (dialog shows Format: DICOM image), browse to location, choose directory, click Open
– or –
Command: open directory-pathname format dicom

...where directory-pathname can also contain wildcards to lessen typing, and should be enclosed in quotes if it includes spaces. Each image stack that is found will be opened as a separate model.

The open command verbose option shows detailed debugging information for DICOM input.

Some DICOM files contain time series that can be played with the time slider shown automatically by ChimeraX or with the vseries command.

Some DICOM files contain segmentations in the form of binary maps (values 0,1); these can be displayed as an isosurface, for example: volume #1.3 style surface level 1
