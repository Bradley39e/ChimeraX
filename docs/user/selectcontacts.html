Select Contacts

Choosing Select... Contacts from the ChimeraX menu opens the Select Contacts dialog, with sections:

Chains
Atomic

Depending on the options used, the corresponding command is contacts, interfaces, or select. See also: Contacts, menu Select... Zone, zone atom specification

OK performs the selection and closes the dialog, whereas Apply performs the selection without closing the dialog. Close closes the dialog without doing anything. Help shows this page in the ChimeraX browser.

Chains

Select chain-chain contacts by choosing two chains, indicating whether to select the contacting residues in both chains or only the first chain, and specifying how the contacts should be identified. Either of the following measures can be used:

  • Buried surface area (default)
    • Contacting chains bury at least [SASAc] (default 100.0 Å2)
    • Contacting residues bury at least [SASAr] (default 15.0 Å2)
    – or –
  • Residues with atoms within [dist] (default 3.5 Å)

Only protein and nucleic acid chains are considered, not ligands, ions, or solvent. To identify contacts of those other types of residues, use the Atomic section.

[back to top: Select Contacts]

Atomic

Select atomic contacts of the current selection based on either the VDW overlap or the distance between atomic centers, with additional options to limit the scope of selection (details...) and to show the full Contacts tool.
